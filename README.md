# Duty::Git

git specific tasks for the duty task manager

## Installation

Clone this repository and add `path/to/duty-git/lib/duty/tasks` to your `.duty.yml`

```
tasks:
  git: path/to/duty-git/lib/duty//tasks
```

## Usage

```
duty
```
